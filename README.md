# Node Project
- author: Shizuka (Izzy) Sato

## Technologies
- Node.js
- Jest
- Redis cache
- MongoDB mongoose

## How to Start
1. 'npm start' in terminal
2. 'brew services start redis' to start redis
3. listening to the port 8080

## To Stop Redis
'brew services stop redis' 

## Test
- Jest
- 'npm test' to start
